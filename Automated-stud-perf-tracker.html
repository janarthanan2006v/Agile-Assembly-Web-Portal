FaceCheck In

A smart, AI-powered attendance tracker that uses facial recognition for secure, contactless student check-ins.

Key Features

Face Enrollment

Register faces securely via webcam. Data is stored locally and processed for recognition.

Real-Time Detection

Live face scanning automatically marks attendance with instant visual feedback.

Attendance Logs

Get daily reports with timestamps and names. Export data for faculty insights.

Works Offline

Designed for classroom environments. No internet needed for core detection to work.

Technology Stack

Next.js Firebase Tailwind CSS TypeScript Python Genkit
